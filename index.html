Welcome to the matrix-sdk-android documentation!

This pages list the complete API that this SDK is exposing to a client application.

We are still building the documentation, so everything is not documented yet.

A few entry points:

  • org.matrix.android.sdk.api.Matrix: The app will have to create and manage a org.matrix.android.sdk.api.Matrix object.

  • From this org.matrix.android.sdk.api.Matrix object, you will be able to get various services, including the org.matrix.android.sdk.api.auth.AuthenticationService.

  • With this org.matrix.android.sdk.api.auth.AuthenticationService you will be able to get an existing org.matrix.android.sdk.api.session.Session, or create one using a org.matrix.android.sdk.api.auth.login.LoginWizard or a org.matrix.android.sdk.api.auth.registration.RegistrationWizard, which will finally give you a org.matrix.android.sdk.api.session.Session.

  • From the org.matrix.android.sdk.api.session.Session, you will be able to retrieve many Services, including the org.matrix.android.sdk.api.session.room.RoomService.

  • From the org.matrix.android.sdk.api.session.room.RoomService, you will be able to list the rooms, create a org.matrix.android.sdk.api.session.room.Room, and get a specific org.matrix.android.sdk.api.session.room.Room.

  • And from a org.matrix.android.sdk.api.session.room.Room, you will be able to do many things, including get a org.matrix.android.sdk.api.session.room.timeline.Timeline, send messages, etc.

Please read the whole documentation to learn more!

This pages list the complete API that this SDK is exposing to a client application.

We are still building the documentation, so everything is not documented yet.

A few entry points:

  • Matrix: The app will have to create and manage a Matrix object.

  • From this Matrix object, you will be able to get various services, including the AuthenticationService.

  • With this AuthenticationService you will be able to get an existing Session, or create one using a LoginWizard or a RegistrationWizard, which will finally give you a Session.
